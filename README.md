# 💈 Barbearia Jé — Site Institucional

Site **elegante e responsivo** para a Barbearia Jé, em Diadema-SP. Apresenta os serviços, uma galeria de trabalhos e canais de contato diretos, com agendamento por WhatsApp em um clique.

🔗 **Demo ao vivo:** https://barbearia-je.vercel.app

## ✨ Características

- 🎨 **Design moderno:** _glass morphism_, gradientes dinâmicos e animações suaves
- 📸 **Galeria de fotos:** carrossel automático com espaços para os trabalhos
- 📱 **Contato direto:** botão flutuante de WhatsApp, link para telefone e Google Maps
- 📲 **100% responsivo:** funciona bem em celular, tablet e computador

## 🛠️ Tecnologias

- **HTML5, CSS3 e JavaScript** puro
- Layout responsivo (mobile-first)
- Deploy contínuo na **Vercel** (todo push atualiza o site automaticamente)
