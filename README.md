## Deploy

Este site foi deployado automaticamente na Vercel. Qualquer push neste repositório atualizará o site automaticamente.

### Para fazer alterações:

1. Clone o repositório
2. Faça suas mudanças
3. Commit e push
4. A Vercel fará o deploy automaticamente

## Adicionar Fotos

Para adicionar mais fotos no carrossel:
1. Abra o site
2. Clique em qualquer espaço vazio da galeria
3. Selecione a foto no seu computador

A foto será salva no seu navegador automaticamente!
